A modern **Quick Commerce Platform** inspired by leading instant delivery companies such as **Zepto**, **Blinkit**, **Instamart**, **Gopuff**, and **Getir**.

This project demonstrates how to build a **production-ready**, **AI-powered**, and **scalable** hyperlocal commerce platform with customer apps, delivery management, warehouse operations, dark stores, analytics, and real-time tracking.
